# 人体解剖及生理学期末小白通关系统

这是一个面向期末突击复习的静态网页复习系统，目标不是系统学习整本书，而是帮助零基础或时间紧张的同学快速抓住高频考点、按步骤复习、刷题、自测和考前冲刺。

网页可直接部署到 GitHub Pages，也可以下载后本地打开 `index.html` 使用。

## 在线使用

如果已开启 GitHub Pages，访问地址通常是：

```text
https://你的用户名.github.io/仓库名/
```

手机端也可以打开。建议在手机浏览器中选择“添加到主屏幕”，之后可以像 App 一样使用。

## 本地使用

下载本仓库后，直接双击：

```text
index.html
```

不需要安装服务器，不需要联网即可查看主要复习内容。

## 功能内容

- 首页：按 1 天、3 天、7 天给出具体复习路线
- 考试内容地图：按考试优先级排列章节
- 章节学习卡片：每章都有小白解释、必背句、易混点、图示、自测题和 checklist
- 章节真题题库：按章节整合原始复习题/自测题/例题，已去重
- 高频标注：题库中高频考点会单独标出
- 答案与解析：题目支持点击展开答案、解析、易错提醒和答题模板
- 总自测区：包含单选、判断、填空、名词解释和简答题
- 考前 2 小时冲刺区：最后必背句、易混点、简答模板和考试提醒
- 进度保存：章节 checklist 和完成状态会保存在浏览器 localStorage
- 打印支持：打印时自动隐藏导航栏、按钮和进度条，只保留正文内容

## 资料依据

内容基于课程 PPT、期末例题、自测题、复习题，以及整理生成的章节题库。

题库答案分为两类：

- **原题答案**：原始资料中明确出现括号答案、定义解释或参考答案时，直接提取。
- **参考答题方向**：原始资料没有显式答案时，依据课程 PPT、例题高频点和公开教学资料整理答题方向、解析和易错提醒。

请注意：如果老师给过标准答案，以老师答案为最高优先级。

## 如何更新内容

如果只是更新网页内容，通常替换这几个文件即可：

```text
index.html
style.css
script.js
.nojekyll
```

如果使用 GitHub 网页上传：

1. 进入仓库首页
2. 点击 `Add file`
3. 点击 `Upload files`
4. 拖入新版文件
5. 点击 `Commit changes`
6. 等待 GitHub Pages 自动刷新

如果手机端仍显示旧版，可以在网址后加：

```text
?v=新数字
```

例如：

```text
https://你的用户名.github.io/仓库名/?v=4
```
